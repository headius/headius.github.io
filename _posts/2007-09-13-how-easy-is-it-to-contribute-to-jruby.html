--- layout: post title: How Easy Is It To Contribute To JRuby? date: '2007-09-13T02:43:00.000-07:00' author: headius tags: modified_time: '2011-01-25T21:44:32.527-08:00' blogger_id: tag:blogger.com,1999:blog-4704664917418794835.post-2205950002493915130 blogger_orig_url: http://blog.headius.com/2007/09/how-easy-is-it-to-contribute-to-jruby.html --- Answer: Very Easy!

Get the Code
svn co http://svn.codehaus.org/jruby/trunk/jruby
cd jruby
ant

Run Your Built JRuby
export PATH=$PATH:$PWD/bin
jruby my_script.rb

Install Gems
gem install somegem
OR
jruby -S gem install somegem

Run the Benchmarks
jruby -J-server -O test/bench/bench_method_dispatch.rb
(-J-server uses the "server" JVM and -O disables ObjectSpace)


Build and Test Your Changes
ant clean test

Look For Bugs To Fix or Report Your Own

JRuby JIRA (bug tracker)


Join the Mailing Lists

JRuby Mailing Lists


Join Us on IRC

Channel #jruby on FreeNode IRC
